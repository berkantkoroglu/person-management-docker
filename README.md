# Person Management System

This project is a full-stack web application built with React, Express.js, PostgreSQL, Docker, and Docker Compose.

## Features

- Add new person
- View all people
- Update existing person
- Delete person

## Technologies

- React
- Node.js / Express.js
- PostgreSQL
- Docker
- Docker Compose

## API Endpoints

- GET /api/people
- GET /api/people/:id
- POST /api/people
- PUT /api/people/:id
- DELETE /api/people/:id

## Run the Project

```bash
docker compose up --build
